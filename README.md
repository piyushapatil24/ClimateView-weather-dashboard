🌤 ClimateView
ClimateView is a weather dashboard that I built to strengthen my React skills and gain hands-on experience working with REST APIs. The application provides real-time weather information, a 5-day forecast, and location-based weather updates through a clean, responsive, and user-friendly interface.

🚀 Features
- 🔍 Search weather by city name
- 📍 Get weather for your current location
- 🌡 View real-time temperature and weather conditions
- 📅 5-day weather forecast
- 💧 Humidity, Wind Speed, Pressure & Visibility
- ☁ Cloud coverage and "Feels Like" temperature
- 🌙 Dark & Light Mode
- 🖥 Responsive design for desktop, tablet, and mobile
- ⚡ Fast API integration using Axios
- 🚨 Loading state and error handling
- 🔎 City autocomplete suggestions

🛠 Built With
- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- Axios
- OpenWeather API

⚙️ Getting Started
Run the project
```bash
npm run dev
```
